# Como Adicionar Imagens ao Site ManhuáWorld

Para que o site tenha uma aparência profissional, você precisará substituir os arquivos de placeholder por imagens reais. Siga estas instruções:

## Imagens Necessárias

1. **hero-bg.jpg** - Imagem de fundo para a seção hero (tela inicial)
   - Tamanho recomendado: 1920x1080px
   - Deve ser uma imagem que represente bem o universo dos manhuás
   - Sugestão: uma arte de manhuá com boa resolução ou uma composição de várias capas

2. **manhua-example.jpg** - Exemplo de arte de manhuá
   - Tamanho recomendado: 800x600px
   - Deve mostrar claramente o estilo artístico típico dos manhuás

3. **manhua-history.jpg** - Imagem histórica de manhuás
   - Tamanho recomendado: 800x600px
   - Pode ser uma imagem de manhuás antigos ou uma linha do tempo visual

## Onde Encontrar Imagens

- **Sites oficiais** das editoras e plataformas de manhuás
- **Lojas online** que vendem manhuás (para capas)
- **Bancos de imagens** como Unsplash, Pexels ou Pixabay (para imagens genéricas)
- **Sites de fãs** com permissão para uso não-comercial

## Como Substituir as Imagens

1. Renomeie suas imagens conforme os nomes acima
2. Coloque-as na pasta `img/` do projeto
3. Substitua os arquivos existentes
4. Verifique se as proporções estão adequadas
5. Teste o site no navegador para garantir que tudo está sendo exibido corretamente

## Direitos Autorais

Lembre-se sempre de respeitar os direitos autorais. Use apenas imagens:
- Que você tenha criado
- De domínio público
- Com licenças que permitam seu uso
- De obras que você tenha permissão para divulgar

Sempre dê crédito aos artistas originais quando necessário.
